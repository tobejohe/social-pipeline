# Prompt-Änderungslogbuch

Jede Änderung an einer Prompt-Datei wird hier dokumentiert — mit Datum, Datei, Grund und erwartetem Effekt. Das verhindert Prompt-Drift und macht Änderungen nachvollziehbar.

Format:
```
### YYYY-MM-DD: prompts/[datei].md
**Grund:** [Warum geändert?]
**Änderung:** [Was konkret?]
**Erwarteter Effekt:** [Was sollte besser werden?]
```
